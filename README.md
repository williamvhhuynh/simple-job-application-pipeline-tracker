# Job Application Pipeline Tracker 

This project will be a **job application pipeline tracker** that helps users record and manage internship/job applications from start to finish. This project is intended to be used by *students* and other *job seekers* who want a simple, yet structured way to organize their applications to stay on track
and on top of their schedule. 

Features to be include, but not limited to are:
- Store application data including company name, role, dates applied, and current status
- Update application status and add notes
- View and filter applications by status, company, deadlines
- Show summary statistics such as count by status

I am interested in this project as I have custom made Notion databases and excel spreadsheets to try organize my job applications. I want to see how this can be implemented into a graphical user interface and create analytics on application statistics. 

## User Stories
- As a user, I want to be able to add an application to an application pipeline tracker and specify the job title, company, location, date applied, and current status.  
- As a user, I want to be able to view a list of all of the applications I have recorded into the tracker and filter them by status. 
- As a user, I want to be able to update the status of an application.
- As a user, I want to be able to remove an application from the tracker. 
- As a user, I want to be able to see summary statistics such as how many applications are in a current status.
